    Adding a Simple "Scroll to the Top" Button to Your Hugo Site

    Using just HTML and CSS

    With only basic HTML and CSS knowledge, it was great news when I found out that a scroll-to-top button can be easily created without Javascript 1. I went ahead to add it to my current Hugo theme; Anatole.

    The main gist is to create an anchor at the top of the page, and have the link / button go to the anchor when pressed so it “scrolls to the top”.

    Where to put the anchor?

    Since the navigation bar / header bar persists while scrolling in the Anatole theme; instead of adding the anchor id="top-anchor" to header class="header", add it to class="wrapper__main" id="top-anchor".

    Placing the anchor in class="wrapper__main" and also class="wrapper__main wrapper__main--fullscreen" will make the link jump up to the top correctly - otherwise it will not move when the button is pressed. Add this id to baseof.html under layouts/_default.

    If you don’t have a baseof.html file already in your Hugo directory, try to find and copy the default from your theme’s repo.

    Link for button

    1 <a href="#top-anchor"><button class="top-anchor">⇧</button></a>

    Place this link right above </body> in baseof.html. My link text is an up arrow symbol, but you could use an emoji, text or image instead.

    Creating the button using CSS

    Create a css file if it doesn’t already exist.

    assets/css/syntax.css

    Then create a button with your own customizations with the class created earlier (top-anchor). I have used these options for mine:

     1.top-anchor { /*jump to top button, link is in baseof.html */
 2  position: fixed;
 3  bottom: 9px;
 4  right: 9px;
 5  max-width: 39px;
 6  max-height: 39px;
 7  width: 100%;
 8  height: 100%;
 9  border: 3px ridge #2a3031;
10  border-radius: 30px;
11  justify-content: center;
12  background: #353b3c;
13  font-size: 2rem;
14  color: #c6c7c4;
15  font-weight: 500;
16  cursor: pointer;
17  opacity: 50%;
18}
19
20.top-anchor:hover {
21  opacity: 100%;
22  color: #B06969;
23}

    Result

    You should get a button on the bottom right corner like this:

    I’ve made mine 50% opacity to reduce distraction, but upon hovering, it will light up to 100% opacity and change to red. You can see the link points to the anchor.

    Hope this helps!
